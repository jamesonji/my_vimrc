# my_vimrc
Song's Vimrc

## Install Vim-Plug

Instruction: https://github.com/junegunn/vim-plug

#### Quick install
```
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

#### Install plugins

Clone the `.vimrc` file and run
```
:PlugInstall
```
